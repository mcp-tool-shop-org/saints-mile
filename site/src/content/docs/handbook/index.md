Saint's Mile is a frontier JRPG for the adults who loved those games first. This handbook contains the complete design reference for building the game — from product thesis through implementation contracts.

## The Core Promise

A weathered frontier road story with sharp banter, real consequences, and just enough mystery to make the horizon feel haunted. Built for the generation that grew up on 90s classics.

**v1.0.1 — Release Candidate.** Full campaign implemented (Prologue + 15 chapters). Combat engine, standoff system, pressure encounters, state management, save/load, and TUI presentation layer all operational. 151 tests passing.
